## Installation

1. Clone the repository

```bash
git clone https://github.com/incandesc3nce/Mini-Message-Board.git
```

2. Install dependencies

```bash
npm install
```

3. Run the server

```bash
npm start
```

4. Open your browser and go to `http://localhost:3000/`
